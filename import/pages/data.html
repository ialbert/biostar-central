How to share data on Biostars

Users can share arbitrary amount of data from their own computers. A bittorrent client is required. Options include:

  1. Official BitTorrent: http://www.bittorrent.com/
  2. Transmission for Mac and Linux: http://www.transmissionbt.com/
  3. Transmission for WindowsL http://sourceforge.net/projects/trqtw/

The instructions below assume that you have installed a BitTorrent client.

How to download data

  1. Download the torrent file.
  2. Double-click the torrent file (or open it via File-->Open).

How to share data with others

  1. Select "New Torrent" from your bittorrent application
  2. Select the file or the directory that you wish to share
  3. Enter https://www.biostars.org/data/announce into the Tracker Field of the new torrent.
  4. Select save. This will create a so called torrent file.
  5. Go to Biostars and create a new post (or edit an old post) and attach the torrent file that you've created to it.
  6. Important! Leave your computer running! The data will be shared directly from your system so the power management must not allow the computer to go to sleep.

How to delete a torrent

  1. Find the post that you have associated with the torrent and click on the torrent link 
  2. At the bottom you will see a "Delete Torrent" button.
  3. Also stop the torrent in your Bittorrent software.
